# Exemples PDF TerrOir

Ce dossier reçoit les PDF d'exemple générés localement pour vérifier le rendu
visuel des documents producteurs, sans créer de données en base.

Commande :

```bash
npm run pdf:examples
```

Fichiers générés :

- `exports/comptabilite-terroir-exemple-2026.pdf`
- `exports/bilan-annuel-terroir-exemple-2026.pdf`

Les PDF générés ne sont pas versionnés pour éviter de committer des binaires
temporaires à chaque ajustement visuel.
